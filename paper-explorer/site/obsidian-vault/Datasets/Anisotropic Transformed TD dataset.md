# Anisotropic Transformed TD dataset

ATTD is created by elongating [[Topologically Diverse dataset]] structures along the z axis.

## Purpose

It introduces systematic anisotropy into a structurally diverse dataset without relying on the RTP generation mechanism.

## Reported Summary

- sigma(k): 0.20.
- sigma(L): 1.74.
- CNN: R2 0.894, MAE 0.34.
- Directional PH+ECP: R2 0.825, MAE 2.06.
- Non-directional PH+ECP: R2 0.643, MAE 3.33.

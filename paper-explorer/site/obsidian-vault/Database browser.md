# Database browser

The database browser is a static site page for inspecting dataset-level metadata and, later, actual data files.

## First Version

Uses paper-reported metadata:

- dataset counts;
- anisotropy measures;
- methods and metrics;
- source repository link.

## Future Version

After [[Direction-aware TDA repository]] is cloned or mounted, the browser should expose real file manifests, descriptor dimensions, and sample previews.

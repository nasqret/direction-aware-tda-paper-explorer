# Cone-based filtration

Cone-based filtration assigns each material voxel a value based on porosity inside a double cone aligned with the compression direction.

## Paper Parameters

- Cone height: 6.
- Cone base radius: 3.
- Empty voxels receive 1.25.
- Periodic wrapping is used for the cone computation.

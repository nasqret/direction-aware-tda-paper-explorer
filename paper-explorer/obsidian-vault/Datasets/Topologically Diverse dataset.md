# Topologically Diverse dataset

The TD dataset contains statistically isotropic porous structures generated from multiple topology families.

## Structure Families

- Voronoi-based.
- Zeolitic-inspired.
- Diamond-like.
- Cubic-strut.
- Spline-based.

## Reported Summary

- Number of structures: 2375.
- sigma(k): 0.14.
- sigma(L): 1.80.
- CNN: R2 0.976, MAE 0.62.
- Directional PH+ECP: R2 0.836, MAE 1.53.
- Non-directional PH+ECP: R2 0.822, MAE 1.46.
